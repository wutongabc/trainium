要开机自动挂载，把挂载信息写入 `/etc/fstab`，推荐使用 UUID。

先确认磁盘是否有分区：

```bash
lsblk -f
sudo blkid /dev/nvme1n1
```

如果实际文件系统在 `/dev/nvme1n1p1`，就使用该分区的 UUID。编辑：

```bash
sudoedit /etc/fstab
```

加入一行（假设文件系统是 ext4）：

```fstab
UUID=你的UUID  /dev3  ext4  defaults,nofail  0  2
```

确保目录存在并测试：

```bash
sudo mkdir -p /dev3
sudo mount -a
findmnt /dev3
```

如果 `mount -a` 没有报错，重启后会自动挂载。

注意：如果 `lsblk -f` 显示文件系统在 `/dev/nvme1n1p1`，不要写 `/dev/nvme1n1`；应使用分区对应的 UUID 和文件系统类型。
